# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Please read the full text at the link above. In short: be respectful, constructive, and inclusive. Harassment and exclusionary behavior are not tolerated.

## Reporting

If you experience unacceptable behavior, please open a GitHub issue or contact the maintainers directly.
